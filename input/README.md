### options
 - ajaxURL 
 - verifiCode
 - labelName 
 - placeHolder 
 - disAbled 
 - maxLen 
 - type 
 - ids 
 - reg
 
#### ajaxURL
ajax请求目录

#### verifiCode
是否启用图片验证码

#### labelName
<label>显示的字符，输入则启用label，不输入则不显示label

#### placeHolder
input中的 placeHolder，默认为不设置

#### disAbled
input的使用/禁用，默认为使用

#### maxLen
input的 maxlength属性，默认为20

#### type
input 正则验证类型
- userNameReg 用户名
- mobilePhoneReg 移动电话
- emailReg 邮箱地址
- passwordReg 密码
- fixedPhoneReg 固定电话
- idCardReg 身份证号
- imgVerifiReg 图片验证码
- phoneVerifiReg 手机或邮箱验证码

#### ids
input的id,(必填)

#### reg
是否自定义正则规则
